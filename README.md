# NLP Hands-On: Práticas de Processamento de Linguagem Natural
Este repositório contém uma série de atividades práticas (Hands-On) desenvolvidas para a disciplina de Processamento de Linguagem Natural (NLP). É aplicar as técnicas de NLP aprendidas em sala de aula de forma prática.
